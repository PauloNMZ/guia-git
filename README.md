# Guia Git na Prática

Guia em HTML/CSS com roteiro didático de Git em português, incluindo conceitos, analogia dos correios, fluxo de trabalho, comandos essenciais, resolução de conflitos e boas práticas.

## Estrutura
- guia-git-pratica.html: página principal com conteúdo e scripts de navegação/abertura de seções.
- styles.css: estilos do guia separados do HTML.

## Como usar
1. Abra guia-git-pratica.html no navegador (duplo clique ou start guia-git-pratica.html no Windows).
2. Navegue pelas abas para ver cada seção.

## Observações
- Arquivo salvo em UTF-8 com BOM para evitar problemas de acentuação.
- Não há dependências externas; basta abrir o HTML.
